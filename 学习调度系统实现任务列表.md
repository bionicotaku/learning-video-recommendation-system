# AI Agent 勾选规则

为了让 agent 真正“做完一项划掉一项”，建议你在任务执行时要求它遵守下面这组规则：

## 规则 1

每次只允许处理**一个子任务标题**，禁止跨任务混做。

## 规则 2

只有满足该子任务表格最后一行“收束验收”，才允许把标题从：

`## [ ] 子任务 xx：...`

改成：

`## [x] 子任务 xx：...`

## 规则 3

如果某个子任务未通过验收，不允许继续下一个子任务。

## 规则 4

每完成一个子任务，必须附带：

- 变更文件列表
- 运行过的命令
- 测试结果
- 是否存在遗留问题

## 规则 5

如果某个子任务发现设计稿与现实实现冲突，必须停在该任务，不允许 agent 自行改大方向。
