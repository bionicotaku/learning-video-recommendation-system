## 服务说明

本应用提供基于短视频内容的语言学习体验，包括视频播放、字幕展示、单词和学习单元解释、练习题、观看进度、学习进度、词书或学习目标集合，以及基于学习状态和内容索引生成的推荐。

当前 MVP 不包含用户上传视频、外部链接处理、公开发布内容、社区互动、付费订阅、广告投放或退款功能。如果未来增加这些能力，我们会在产品和相关文档中另行说明。
